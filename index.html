Website Sistem Rekomendasi Tempat Wisata di Jawa Timur

November 2021 - Desember 2021

"Capstone Project" saya bersama tim saya pada program Bootcamp Pengembang Machine Learning Dan Front-End Web, dimana kami merancang sebuah website sederhana, sistem akan memberikan rekomendasi destinasi wisata yang akan dikunjungi khususnya di provinsi Jawa Timur. Perancangan sistem dimulai dari mendapatkan Dataset mentah dari Kaggle Dataset. Dataset ini berisi nama, kategori, kota, harga, rating, dan review tempat wisata, kemudian dari dataset tersebut dibuat model machine learning untuk menyajikan model yang dapat merekomendasikan destinasi wisata berdasarkan kemiripan kategori tempat wisata yang dicari, kemudian merancang antarmuka website sederhana lalu mendeploy model machine learning yang sudah dibuat menjadi website.
Tugas:

  • Membangun model di Google Colaboratory menggunakan pendekatan Content Based Filtering dan fungsi TF-IDF Vectorizer
  • Melakukan preprocessing data menggunakan Delete Unnecessary Columns dan Missing Values Imputation
  • Melakukan perhitungan derajat kesamaan (Similarity Degree) antar tempat wisata dengan teknik Cosine Similarity
  • Mengevaluasi model machine learning menggunakan metrik Precision

Dilaut - Aplikasi Berbasis Android

April 2021 - Juni 2021

Proyek ini dilaksanakan sebagai "Capstone Project" saya bersama tim saya pada program Bootcamp Cloud Computing. Terdiri dari 6 orang yang terbagi dari 3 Path berbeda (@2 orang tiap path), kami membuat sebuah aplikasi android bernama "DILAUT". Aplikasi ini menampilkan katalog ikan serta harga ikan, yaitu harga bulan ini dan prediksi harga ikan untuk bulan depan. Dengan "DILAUT" diharapkan dapat memberikan transparansi harga ikan sehingga nelayan tidak terjebak dengan harga yang diberikan oleh tengkulak, khususnya di Kabupaten Gresik, Jawa Timur. Perancangan dimulai dari mendapatkan Dataset mentah dari Dinas Perikanan Gresik, Jawa Timur. Sebagian besar data dihitung setiap bulan. Dataset ini berisi nama ikan dan harga setiap bulan mulai dari 2016 sampai 2020, kemudian dari dataset tersebut tim Path Machine Learning membangun model dengan TensorFlow, menggunakan pipeline data (LSTM) untuk menyajikan model yang dapat memprediksi harga, dan melakukan preprocessing data menggunakan Missing Values Imputation. Selanjutnya, tim Path Mobile Development membuat UI/UX dari splash screen, aktivitas yang sesuai dengan Google Material Design, kemudian integrasikan data JSON secara online menggunakan API dari Google Cloud Platform (GCP) menggunakan metode JLoop/Retrofit, menggunakan MVVM (Model View ViewModel) Arsitektur agar coding dapat diubah menjadi lebih baik. Lalu saya bersama 1 orang tim saya yang termasuk dalam Path Cloud Computing bertugas:

  • Membuat proyek di GCP dan membagikan role owner kepada tiap orang di tim Cloud Computing
  • Membuat Bucket untuk mengupload pengujian set data untuk model machine learning
  • Membuat Cloud Function untuk menyediakan API untuk Android menggunakan model machine learning

JAJAN (Jasa Jahit Online) - Aplikasi Berbasis Android sebagai Solusi Penjahit Rumahan

Oktober 2020 - Januari 2021

Pada saat saya berada di Semester 5 sebagai tugas akhir untuk mata kuliah Proyek Kreativitas Teknologi Informasi (PKTI) dan juga sebagai usulan Program Kerativitas Mahasiswa (PKM) Tahun 2020, saya bersama tim saya merancang sebuah aplikasi android bernama "JAJAN". Aplikasi ini dibuat untuk mempermudah pelanggan dalam menemukan penjahit dan memesan pakaian dengan berbagai macam pilihan yang disediakan. "JAJAN" dilengkapi dengan berbagai fitur untuk mempermudah pemesanan pakaian, antara lain: Chat and Call, Order, Catalog, Tips and Tutorial, dan lain-lain. Perancangan aplikasi dilakukan menggunakan bahasa pemrograman Kotlin pada software Android Studio.
Tugas:
